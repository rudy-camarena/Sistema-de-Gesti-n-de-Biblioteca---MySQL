# 📚 Sistema de Gestión de Biblioteca - MySQL

## 🛠️ Tecnologías utilizadas
**Motor de base de datos:** MariaDB / MySQL
**Entorno:** VS Code
**Lenguaje:** SQL 

## 📋 Características técnicas
**Normalización:** Separación de entidades en tablas independientes (autores y libros).
**Integridad Referencial:** Implementación de Claves Primarias (`PK`) y Claves Foráneas (`FK`) para asegurar la coherencia de los datos.
**Consultas avanzadas:** Uso de `JOIN` para relacionar la información de ambas tablas.
